# v0.2.6 Manual Test Suite - FerriteEditor Release

This document provides a comprehensive manual test checklist for the new FerriteEditor custom text editor. v0.2.6 is a major release replacing egui's TextEdit with a fully custom implementation featuring virtual scrolling.

## Pre-Test Setup

- [x] Build release: `cargo build --release`
- [x] Have test files ready:
  - Small file: < 100KB markdown
  - Medium file: 1-4MB text/markdown
  - Large file: 10MB+ text file
  - Various encodings: UTF-8, UTF-16, Latin-1, Shift-JIS
  - File with box drawing characters: `test_md/test_box_drawing.md`
  - CJK content files: `test_md/test_chinese.md`, `test_md/test_japanese.md`, `test_md/test_korean.md`
